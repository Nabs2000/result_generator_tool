ResultsFile Generator Tool

Please follow the steps below to generate the ResultsFile:

  1. Download the Grades from the Portal in an excel file
  2. Use "Choose File" button and find the excel file from Step 1.
  3. Click the "Upload" Button
  4. The Results File will be generated in your "Downloads" directory
